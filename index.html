My Skill

PHP

PHP adalah singkatan dari "Hypertext Preprocessor." Ini adalah bahasa pemrograman yang sering digunakan untuk pengembangan situs web dan aplikasi web. PHP adalah bahasa pemrograman sisi server, yang berarti kode PHP dieksekusi pada server web untuk menghasilkan halaman web yang dikirimkan ke browser pengguna.

Java

Java adalah bahasa pemrograman yang kuat, berorientasi objek, dan sering digunakan untuk pengembangan perangkat lunak berbasis aplikasi, permainan, aplikasi seluler (Android), aplikasi web, sistem tertanam, dan banyak lagi.

Laravel

Laravel adalah kerangka kerja (framework) pengembangan aplikasi web berbasis PHP yang sangat populer. Laravel dirancang untuk mempermudah pengembangan aplikasi web dengan menyediakan berbagai alat dan fitur yang kuat.

Html

HTML, atau singkatan dari "Hypertext Markup Language," adalah bahasa markup yang digunakan untuk membuat dan mengatur konten pada halaman web. HTML berfungsi sebagai kerangka dasar untuk membangun struktur dan tampilan halaman web.

Github

GitHub adalah platform pengembangan perangkat lunak yang sangat populer yang digunakan oleh pengembang di seluruh dunia untuk mengelola dan berkolaborasi dalam proyek perangkat lunak.

JavaScript

JavaScript adalah suatu bahasa pemrograman tingkat tinggi dan dinamis. JavaScript populer di internet dan dapat bekerja di sebagian besar penjelajah web populer seperti Google Chrome, Internet Explorer, Mozilla Firefox, Netscape dan Opera. Kode JavaScript dapat disisipkan dalam halaman web menggunakan tag script.

Node.js

Node.js adalah platform perangkat lunak yang dirancang oleh Ryan Dahl. Node.js adalah runtime untuk lingkungan JavaScript di luar peramban web yang dibangun di atas mesin JavaScript V8.
